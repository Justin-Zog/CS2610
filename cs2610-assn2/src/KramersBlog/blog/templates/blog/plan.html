Phase 1: Design

Deliver

This project does not require us to use a lot of functions/write our own functions
Instead we are using the build in structure of Django to apply our html files.
This will include making sure Django's framework is understood and I follow the 'rules' set up by that.
The only functions we really have to write happen in the models.py file. Here we take care of requests and serving up a webpage.
