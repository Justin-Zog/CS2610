Phase 1: Design

Deliver

Main Page

The main page will consist of a title that is rendered using a `h1` or `h2` tag. It will also have a byline the gives the name of the blog post and author with the date it was posted. It will also include a body that is more than 300 words long. Lorem Ipsum nonsense is okay to fill the `<p>` with.

About Me

  • Includes a photo of the client and a 3 paragraph (<p>) sketch of the clients life. (Lorem Ipsum nonsense is okay here too).
  • Also includes 2-3 other pictures that showcase how awesome the client is. These images must be actual image files, not links to some image hosted elsewhere.

Tech Tips with and withous CSS

These pages will look almost identical but one will use CSS and the other will only use HTML.

  • Includes a cheatsheet of html and css used to create the site.
  • The elements included in the cheatsheet will be HTML Elements, HTML Character Entitites, CSS Selectors, and CSS Properties.
