Some HTML Elements Used on this Blog

<p>

  • name: Paragraph Element
  • purpose: Represents and displays text in a paragraph format
  • display: Whatever text is included between its tags
  • default element display type: block-level

<div>

  • name: Content Division Element
  • purpose: A container to help with the organization and flow of content on a page.
  • display: It will display any elements contained within its tag.
  • default element display type: block-level

<h1-h6>

  • name: Header Element
  • purpose: Creates and displays header text.
  • display: usually larger header text
  • default element display type: block-level

<ul>

  • name: Unordered List Element
  • purpose: Creates unordered list (like the one you are reading)
  • display: A nice bulletted list
  • default element display type: block-level

CSS Selectors Used On This Blog

.rosepink class selector

  This is a class selector that changes text to rosepink This is used on the last paragraph of the "blog" page.

  You can apply a class selector like so: " <p class="rosepink"> "

#freespirit id tag

  id tags can be applied to vairious things in html. All the #freespirit tag does on my blog is change text to a yellowish color.

  You can apply an id tag like so: " <ul id="freespirit"> "

li:nth-child(odd) css selector

  This nth child selector puts a light blue border around every other li element on my blog.

  You can apply an nth child selector in a CSS stylesheet like so: " li:nth-child(odd) { border: 4px solid #B0E4EA; } "

div type selector

  This CSS type selector makes every "div" tag element have a certain margin, border, padding, and background-color specifications.

  You can apply a type selector like so: " div { background-color: #53CBCF; margin: 20px 35px; border: 7px double #0DA3BA; padding: 10px 10px 10px 10px; }

Some CSS Properties Used On This Blog

margin

  This property determines the margin that an html element will have.

  Possible Values: 0 auto; 20px 35px;

color

  This property determines the color that an html element will have.

  Possible Values: black; white; green;

font-family

  This property determines the font family that a text element will have.

  Possible Values: Georgia, serif; san-serif;
